```py
def pairs(self, page: int=1, blockchain: str | None=None, exchange: str | None=None) -> Pairs:
```
Get the list of pairs supported by the API

## Parameters

 - [page](#page): Each request has a maximum of 100 results separated by page, defaults to 1
 - [blockchain](#blockchain): The blockchain from which you want to get the pairs, defaults to None
 - [exchange](#exchange): The exchange from which you want to get the pairs, defaults to None

## Returns

<CodeBlock language="python">
    <a href="/docs/python-sdk/models/pairs">Pairs</a>
</CodeBlock>

The list of pairs supported by the API. It returns token addresses,
blockchain, exchange and the fee that the pair has.

The fee may vary depending on the exchange used.


## Example

### Usage

```py
import asyncio

from BlockchainAPIsSync import BlockchainAPIsSync

async def print_pairs():
    # Create the BlockchainAPIsSync instance
    # You can additionaly add an API key if you want
    blockchain_apis_sync = BlockchainAPIsSync()
    # Get the list of pairs supported by the API
    pairs = await blockchain_apis_sync.pairs(
        page=1
        blockchain=ethereum
        exchange=uniswapv2_ethereum
    )
    print(pairs)
    # We need to close our instance once we are done with BlockchainAPIs
    await BlockchainAPIsSync.close()

asyncio.run(print_pairs())
```

## Exceptions

- [BlockchainNotSupportedException](/docs/python-sdk/exceptions/blockchain-not-supported-exception): When an invalid blockchain id is given
- [ExchangeNotSupportedException](/docs/python-sdk/exceptions/exchange-not-supported-exception): When an invalid Exchange id is given
- [InvalidPageException](/docs/python-sdk/exceptions/invalid-page-exception): When you give an invalid page number
