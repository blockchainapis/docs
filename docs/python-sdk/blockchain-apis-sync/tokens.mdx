```py
def tokens(self, page: int=1, blockchain: str | None=None) -> Tokens:
```
Get the list of supported tokens

## Parameters

 - [page](#page): Each request have a limit of 100 data separated per pages, defaults to 1
 - [blockchain](#blockchain): The blockchain on which you want to get the tokens, defaults to None

## Returns

<CodeBlock language="python">
    <a href="/docs/python-sdk/models/tokens">Tokens</a>
</CodeBlock>

The list of supported tokens ordered by market cap in a descending order.
The market capitalization is in dollars, it can be null if the liquidity available for the given token is lower than 1000$.
The market capitalization is based on USDT.

The market capitalization of each token is computed as follow:
average worth of token in liquidity pools * total token supply


## Example

### Usage

```py
import asyncio

from BlockchainAPIsSync import BlockchainAPIsSync

async def print_tokens():
    # Create the BlockchainAPIsSync instance
    # You can additionaly add an API key if you want
    blockchain_apis_sync = BlockchainAPIsSync()
    # Get the list of supported tokens
    tokens = await blockchain_apis_sync.tokens(
        page=1
        blockchain=ethereum
    )
    print(tokens)
    # We need to close our instance once we are done with BlockchainAPIs
    await BlockchainAPIsSync.close()

asyncio.run(print_tokens())
```
