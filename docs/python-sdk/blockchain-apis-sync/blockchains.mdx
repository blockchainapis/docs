Get the list of blockchains supported by the API
## Returns

<CodeBlock language="python">
    List[<a href="/docs/python-sdk/models/Blockchain">Blockchain</a>]
</CodeBlock>

The list of the blockchains supported by the API.
Using this method, you can find the id of the blockchain that you can use for
other function calls.


## Example

### Usage

```py
import asyncio

from BlockchainAPIsSync import BlockchainAPIsSync

async def print_blockchains():
    # Create the BlockchainAPIsSync instance
    # You can additionaly add an API key if you want
    blockchain_apis_sync = BlockchainAPIsSync()
    # Get the list of blockchains supported by the API
    blockchains = await blockchain_apis_sync.blockchains()
    print(blockchains)
    # We need to close our instance once we are done with BlockchainAPIs
    await BlockchainAPIsSync.close()

asyncio.run(print_blockchains())
```
