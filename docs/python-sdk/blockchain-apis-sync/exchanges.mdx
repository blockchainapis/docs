```py
def exchanges(self, page: int=1, blockchain: str | None=None) -> Exchanges:
```
Get the list of supported exchanges by the API

## Parameters

 - [page](#page): You can ignore this value for this version of the API., defaults to 1
 - [blockchain](#blockchain): The blockchain from which you want to get the exchanges, defaults to None

## Returns

<CodeBlock language="python">
    <a href="/docs/python-sdk/models/Exchanges">Exchanges</a>
</CodeBlock>

The list of all supported exchange of the API.
You can use the exchange id responded from this for other API calls.


## Example

### Usage

```py
import asyncio

from BlockchainAPIsSync import BlockchainAPIsSync

async def print_exchanges():
    # Create the BlockchainAPIsSync instance
    # You can additionaly add an API key if you want
    blockchain_apis_sync = BlockchainAPIsSync()
    # Get the list of supported exchanges by the API
    exchanges = await blockchain_apis_sync.exchanges(
        page=1
        blockchain=ethereum
    )
    print(exchanges)
    # We need to close our instance once we are done with BlockchainAPIs
    await BlockchainAPIsSync.close()

asyncio.run(print_exchanges())
```
