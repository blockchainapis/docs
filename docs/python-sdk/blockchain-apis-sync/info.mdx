```py
def info(self, blockchain: str, token: str) -> Token:
```
Get information on a specific token

## Parameters

 - [blockchain](#blockchain): The blockchain on which you want to get the information of the token
 - [token](#token): The address of the token that you want to get the informations from

## Returns

<CodeBlock language="python">
    <a href="/docs/python-sdk/models/Token">Token</a>
</CodeBlock>

Informations on a specific token.
This information includes:
- The id of the blockchain that this token is
- The address of the token
- The decimals of it
- The market cap


## Example

### Usage

```py
import asyncio

from BlockchainAPIsSync import BlockchainAPIsSync

async def print_info():
    # Create the BlockchainAPIsSync instance
    # You can additionaly add an API key if you want
    blockchain_apis_sync = BlockchainAPIsSync()
    # Get information on a specific token
    info = await blockchain_apis_sync.info(
        blockchain="ethereum"
        token="0xC02aaA39b223FE8D0A0e5C4F27eAD9083C756Cc2"
    )
    print(info)
    # We need to close our instance once we are done with BlockchainAPIs
    await BlockchainAPIsSync.close()

asyncio.run(print_info())
```

### Example response

## Exceptions

- [BlockchainNotSupportedException](/docs/python-sdk/exceptions/BlockchainNotSupportedException): When an invalid blockchain id is given
- [TokenNotFoundException](/docs/python-sdk/exceptions/TokenNotFoundException): When the given token is not found

## Parameters detailed
### blockchain

The blockchain on which you want to get the information of the token
- type: `str`
- example: `ethereum`

### token

The address of the token that you want to get the informations from
- type: `str`
- example: `0xC02aaA39b223FE8D0A0e5C4F27eAD9083C756Cc2`
