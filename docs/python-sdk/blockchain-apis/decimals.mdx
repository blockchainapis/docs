Get the decimals of the given token

## Parameters

 - [blockchain](#blockchain): The id of the blockchain of the token
 - [token](#token): The address of the token that you want to get the decimals from

## Returns

<CodeBlock language="python">
    int
</CodeBlock>

The decimals of the specific token
## Example

### Usage

```py
import asyncio

from BlockchainAPIs import BlockchainAPIs

async def print_decimals():
    # Create the BlockchainAPIs instance
    # You can additionaly add an API key if you want
    blockchain_apis = BlockchainAPIs()
    # Get the decimals of the given token
    decimals = await blockchain_apis.decimals(
        blockchain="ethereum"
        token="0xC02aaA39b223FE8D0A0e5C4F27eAD9083C756Cc2"
    )
    print(decimals)
    # We need to close our instance once we are done with BlockchainAPIs
    await BlockchainAPIs.close()

asyncio.run(print_decimals())
```

### Example response


```json
16
```
## Exceptions

- [BlockchainNotSupportedException](/docs/python-sdk/exceptions/blockchain-not-supported-exception): When an invalid blockchain id is given
- [TokenNotFoundException](/docs/python-sdk/exceptions/token-not-found-exception): WHen the given token is not found
