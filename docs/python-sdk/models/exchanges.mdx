```python
@dataclass(slots=True, frozen=True)
class Exchanges:
    page: int
    total_pages: int
    data: List[Exchange]
```

## Attributes

### page

The page returned by the request

- type: `int`
- example: `1`

### total_pages

The total amount of pages available

- type: `int`
- example: `42`

### data

The list of exchanges

#### type

<CodeBlock language="python">
    List[<a href="/">Exchange</a>]
</CodeBlock>
