# Get Your API Key

You can use Blockchain APIs in a limited way without an API key. With an API key you can
get more requests per second and per month, this way you can utilise the power of Blockchain
APIs.

In this tutorial, you will learn how you can get an API key for free.

## STEP 1: Create an account

### Verify your email

First, you need to create an account at: https://dashboard.blockchainapis.io/

Specify your e-mail address in the form:

<img loading="eager" alt="Blockchain APIs dashboard (Sign UP)" src="/img/docs/api-key/sign-up.png" />

Once you have put your e-mail address, you need to check your e-mail for a subscription link. The link should look like this:

<img loading="eager" alt="Verification email" src="/img/docs/api-key/verification-email.png" />

By clinking with this page, you should end up in the subscription page:

:::danger
Make sure that the url that you have starts with: https://dashboard.blockchainapis.io/
:::

### Choose your plan

Once you have created your account, you can choose your plan depending on your needs on this page:

<img loading="eager" alt="Choose plan page" src="/img/docs/api-key/choose-plan.png" />

In order to get a free API key, you can choose the `Developer` tier:

<img loading="eager" alt="Choose plan page (Developer tier selected)" src="/img/docs/api-key/choose-plan-developer.png" />

## STEP 2: Get your API key from the dashboard

Once you have selected your plan, you will be redirected to the dashboard:

<img loading="eager" alt="Blockchain APIs dashboard index" src="/img/docs/api-key/dashboard.png" />

In order to see your API key, click on `API Key` or follow this link: https://dashboard.blockchainapis.io/api-key

Your API key should be displayed here:

<img loading="eager" alt="API key" src="/img/docs/api-key/api-key.png" />

You can now just click on the `Copy` button:

<img loading="eager" alt="API key with arrow" src="/img/docs/api-key/api-key-arrow.png" />

:::danger
You should never share your API key with anyone, our support will never ask for your API key.
If you think that your API key is no longer safe, you can reset it by clicking on
the `Reset` button near the `Copy` button.
:::

### Instantiate the main class with the API

<Tabs groupId="programming-language" queryString>
<TabItem value="python" label="Python" default>

```py
from blockchainapis import BlockchainAPIsSync

# Put your API key here
api_key = "<put your API key here>"

blockchain_apis = BlockchainAPIsSync(api_key)

# do some calls
print(blockchain_apis.blockchains())
```

</TabItem>
<TabItem value="python-async" label="Python-Async">

```py
import asyncio

from blockchainapis import BlockchainAPIs

# Put your API key here
api_key = "<put your API key here>"

async def get_blockchains():
    async with BlockchainAPIs(api_key) as blockchain_apis:
        # do you API calls
        print(await blockchain_apis.blockchains())

asyncio.run(get_blockchains())
```
